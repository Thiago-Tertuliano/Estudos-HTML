│           HEADER               │
│      Logo + Menu de Navegação  │

│    Título + Texto + Botão      │

├─────────────────────────────────┤
│           SOBRE                │
│    Descrição do Produto        │
├─────────────────────────────────┤
│         VANTAGENS              │

**Elementos obrigatórios**:
- ✅ Header com navegação fixa

- ✅ Formulário de contato funcional

- ✅ Design responsivo completo

### 📱 Responsividade
```css
/* Mobile-first approach */
.container {
  width: 100%;
  padding: 20px;
}

@media (min-width: 768px) {
  .container {
    max-width: 1200px;
    margin: 0 auto;
  }
}
```
